# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

- **Do not** open a public GitHub issue.
- Instead, report the issue privately using GitHub Security Advisories.

You can report a vulnerability here:
https://github.com/# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

- **Do not** open a public GitHub issue.
- Instead, report the issue privately using GitHub Security Advisories.

You can report a vulnerability here:
https://github.com/CarolFreire01/security-template/security/advisories/new

Example: https://github.com/<your-username>/<repo-name>/security/advisories/new

## Supported Versions

Only the latest version of this project is actively supported with security updates.

## Response Expectations

- Initial response: within **7 days**
- Security fixes will be prioritized based on severity

## Disclosure Policy
